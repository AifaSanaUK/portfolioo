# How to Show Files in VS Code / Cursor

## Quick Methods to Show File Explorer:

### Method 1: Keyboard Shortcut
Press: **`Ctrl + Shift + E`**

### Method 2: Menu
1. Click **View** in the top menu
2. Select **Explorer** (or **File Explorer**)

### Method 3: Sidebar Icon
1. Look for the **folder icon** on the left sidebar
2. Click it to open the file explorer

### Method 4: Command Palette
1. Press **`Ctrl + Shift + P`**
2. Type: **"View: Show Explorer"**
3. Press Enter

## If Files Still Don't Show:

1. **Open the folder:**
   - File → Open Folder
   - Navigate to: `C:\Users\aifas\OneDrive\Desktop\portfolio`
   - Click "Select Folder"

2. **Refresh the explorer:**
   - Right-click in the file explorer
   - Select "Refresh"

3. **Check if folder is collapsed:**
   - Look for small arrows (▶) next to folder names
   - Click them to expand folders
